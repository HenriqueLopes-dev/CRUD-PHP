# CRUD PHP

## Pré-requisitos
- xampp instalado 

## Como baixar o Projeto

```bash
git clone https://github.com/HenriqueLopes-dev/CRUD-PHP.git
cd CRUD-PHP
```

## Configurando o ambiente

### **ATENÇÃO:** para o projeto funcionar corretamente, é preciso mover as pastas da seguinte forma:
- **crud_private** deve ser movido para a pasta raiz do xampp (xampp/)
- **crud_public** deve ser movido para a pasta htdocs do xampp (xampp/htdocs)
  
### Banco de Dados (phpMyAdmin)
- **Acesso via navegador**: [http://localhost/phpmyadmin/index.php](http://localhost/phpmyadmin/index.php)
